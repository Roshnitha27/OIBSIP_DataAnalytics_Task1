# OIBSIP Data Analytics Internship - Task 1

## Project Title
EDA on Retail Sales Data

## Objective
The objective of this project is to perform Exploratory Data Analysis (EDA) on retail sales data to identify customer purchasing patterns, product performance, and sales trends.

## Tools Used
- Microsoft Excel

## Steps Performed
1. Imported and reviewed the dataset
2. Cleaned and prepared the data
3. Analyzed sales performance
4. Identified customer purchasing patterns
5. Created charts and visualizations
6. Generated business insights

## Key Insights
- Identified top-selling products
- Analyzed sales trends
- Evaluated customer purchasing behavior
- Highlighted opportunities for business improvement

## Outcome
The project transformed raw retail sales data into meaningful insights that support data-driven business decisions.
